# Training Report System

## Description
A C# console application that generates reports from training data using LINQ.

## Features
- Soldier performance report
- Training performance report
- General statistics
- Ranking based on average scores

## Technologies
- C#
- LINQ
- Console Application

## Concepts Applied
- Join operations
- GroupBy
- Data projection
- Aggregations (Average, Count, OrderBy)

## Data Source
This project uses an in-memory repository to simulate a database.

## How to Run
bash
dotnet run
